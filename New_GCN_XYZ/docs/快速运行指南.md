## 📊 结果文件清单

运行完成后，在结果文件夹中会有以下7个文件：

### 1. 📋 comparison_table.csv
- **用途**: Excel可以打开的对比表格
- **内容**: 所有算法在不同订单规模下的Makespan和Total Travel Time
- **适合**: 直接复制到论文中作为表格

### 2. 📄 comparison_report.txt
- **用途**: 详细的文本报告
- **内容**: 
  - 每个实验的完整结果
  - 改进百分比
  - 统计分析（平均改进、最大改进、标准差）
- **适合**: 查看详细数据和分析

### 3. 📈 chart_total_travel.png
- **用途**: 总行驶时间对比折线图
- **内容**: 4条曲线（GCN, Best Fit, First Fit, Nearest Neighbor）
- **适合**: 论文中的对比图

### 4. 📈 chart_makespan.png
- **用途**: Makespan对比折线图
- **内容**: 4条曲线显示不同算法的Makespan
- **适合**: 论文中的对比图

### 5. 📈 chart_improvement.png
- **用途**: 改进百分比折线图
- **内容**: GCN相对于最佳基线的改进趋势
- **适合**: 展示算法优势

### 6. 📊 chart_combined.png
- **用途**: 组合对比图（最重要！）
- **内容**: 4个子图
  - (a) 总行驶时间对比
  - (b) Makespan对比
  - (c) GCN改进百分比
  - (d) 总成本柱状图
- **适合**: 类似你中期报告的Figure 4.1，可以直接用于论文

### 7. 📦 raw_results.json
- **用途**: 原始实验数据（JSON格式）
- **内容**: 所有实验的完整数据
- **适合**: 后续分析或重新生成图表
